# clean-architecture-android
A Simple Android Mobile Application which has been implemented using MVVM Clean Architecture.


## Tech Stack

- Clean Architecture
- MVVM Pattern
- Kotlin
- Coroutines, Flow 
- LiveData
- Dagger Hilt
- Retrofit2
- Navigation Architecture Component
